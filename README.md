
# Weather App

A react native weather app made with react native paper

## Features

- Current weather
- local and world news (world news on home page & local news once location selected)


## Run Locally


```bash
  git clone https://github.com/mukulve/Weather-App
  cd Weather-App
  npm install
  npm run start
```
